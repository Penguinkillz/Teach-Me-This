# Teach Me This

Standalone AI personal explainer. Topic or file in → structured explanation + practice questions out.

- **Local:** `uvicorn main:app --reload --port 8000` then open http://localhost:8000
- **Env:** Copy `.env.example` to `.env` and set `PLATFORM_GROQ_API_KEY`
- **Deploy:** Same as any FastAPI app (e.g. Railway, Procfile included).
